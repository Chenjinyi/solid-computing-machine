# huaban-Spider

#### 功能

​	未登录状态搜索花瓣图片，只能搜索前几十张

​	由于花瓣限制，图片URL不允许下载，本爬虫会自动爬取URL保存到文件里面

#### 使用方法

```shell
php -f Spider.php -name
```

[-name]

​	要搜索的关键词
